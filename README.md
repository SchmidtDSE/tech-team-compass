# Schmidt DSE Tech Team Compass

**A team compass is a place for community members, new and old, to align on objectives,
practices, and happenings.**

Please [view the tech team compass website](https://schmidtdse.github.io/tech-team-compass/)!


## Usage

This repo is built with [MySTMD](https://mystmd.org/) using
[pixi](https://pixi.prefix.dev/latest/).

This repo is deployed with GitHub Pages, and PR previews are deployed with ReadTheDocs.


## Contributing

### Building locally

You can build locally with [pixi](https://pixi.prefix.dev/latest/):

```bash
pixi run preview
```


### Adding, moving, or removing a page

When you add, move, or remove a page, the Table of Contents (TOC) needs to be updated.
Ensure you edit `myst.yml` accordingly!
